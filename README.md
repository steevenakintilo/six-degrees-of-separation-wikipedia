# six-degrees-of-separation-wikipedia
six degrees of separation but for wikipedia

Download all wikepedia french version:

https://download.kiwix.org/zim/wikipedia/

File name wikipedia_fr_all_maxi_2026-02

Download folder of real people from this archive:

https://drive.google.com/drive/folders/1UYFh8I5bjSn9fLi4iCDX7ubqZXSqkEkV?usp=sharing

Put the folder inside src folder.

Download all the modules required for the bot to work using this command:

```bash
    pip install -r requirements.txt
```

To run the code just do:

```bash
    python main.py
```
